# Criterios de Aceptación

## Funcionales
- Script tipo strategy
- Genera entradas/salidas
- Permite backtesting
- Soporta múltiples temporalidades
- Soporta multiples estrategias

## Métricas
Debe permitir evaluar:
- win rate
- profit
- drawdown

## Flexibilidad
- parámetros configurables
- filtros activables/desactivables

## Usabilidad
- código claro
- inputs organizados
- visualización entendible

## Calidad
- mantenible
- modular
- preparado para iteraciones

## Futuro
Base sólida para:
- optimización
- dashboards
- nuevas funcionalidades
