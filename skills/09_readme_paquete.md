# README del Paquete para Claude

Este paquete contiene la base documental del proyecto para desarrollar una estrategia en TradingView usando Pine Script.

## Archivos incluidos
- `00_project_overview.md`
- `01_strategy_goal.md`
- `02_pinescript_requirements.md`
- `03_backtesting_requirements.md`
- `04_parameters_and_inputs.md`
- `05_analysis_workflow.md`
- `06_acceptance_criteria.md`
- `07_claude_master_prompt.md`
- `08_estilo_visual_entradas_salidas.md`

## Uso recomendado
Cargar todos los archivos como contexto del proyecto y luego dar una instrucción como:

`Usa todos los markdown adjuntos como especificación del proyecto. No inventes reglas de trading. Espera mi lógica exacta y conviértela en una estrategia Pine Script modular, compatible con backtesting, expansión de parámetros y visualización avanzada de entradas/salidas.`

## Enfoque del paquete
Este paquete busca que Claude:
- mantenga una arquitectura consistente,
- entienda el objetivo del proyecto,
- prepare la estrategia para backtesting,
- respete el estilo visual deseado,
- y permita evolución continua del sistema.
